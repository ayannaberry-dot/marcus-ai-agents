New SOP drafted · Weekend showing requests on Marcus's listings

Created Apr 28, 2026 · Author: Marcus (captured by voice) · Owner: Javier

SOP body

When a buyer agent requests a showing on one of Marcus's active listings for a Saturday or Sunday:

1. Confirm the request within 2 hours, even if you can't yet schedule. "Got it, working on confirming with the seller now" is the standard ack.

2. Check seller access rules in the listing folder. If sellers have blackout windows (e.g., Sunday morning church, Saturday before 10 AM), don't propose those.

3. Default cadence: 30-min slots. 15-min buffer between back-to-back requests on the same property.

4. Confirm with seller by their preferred channel (logged in their client file). Tina Tran prefers text; Henrik Liu prefers email; Costa family wants a phone call for anything weekend-day-of.

5. Once confirmed:
– Send buyer agent confirmation in their preferred channel (referral network has this)
– Update lockbox / showing-app system
– Log on the listing's showing calendar in 09_data
– Flag in Daily Debrief if showing is Sat AM (Marcus likes to be aware of weekend traffic on listings)

6. Day-of:
– Send the buyer agent a 60-min reminder
– Capture post-showing feedback within 4 hours via the showings-host helper
– Roll up to seller in Friday digest

Edge cases:
– Buyer agent wants <24 hours notice → Tina/seller confirmation by phone, not text. If you can't reach seller in 30 min, decline and offer next available.
– Multiple buyer agents requesting the same window → first-come, first-confirmed. Don't double-book.
– Open house weekend → buyer-agent appointments by appointment only outside open house hours; during OH, walk-ins welcome but capture name + agent.
